# LINK ME

## OBJETIVO DO PROJETO
O objetivo desse projeto é desenvolver um Link Tree, no qual as pessoas que acessarem essa página poderam ter o acesso facilitado a todos
os meu meios de contato, redes sociais e portifolio.
